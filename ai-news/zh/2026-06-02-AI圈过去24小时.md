# AI 圈过去 24 小时 · 2026-06-02

> 发布日期:2026-06-02 · 类型:AI 热点日报

---

今天最值得 Tony 关心的核心信号,是热点几乎全压在 agent 和 skill 这一层——不再是「又出了个更强的模型」,而是「怎么让 agent 真正理解你、能记住、能被一个人驾驭着交付一整套产品」。模型卷了一年参数,这两天却集体转向卷「系统」:有记忆、可治理、能独立干完一件事。而撑起这套生态的 Anthropic,同一天悄悄递交了 IPO 草案。一句话:**AI 正在从「一次性对话工具」,长成「有记忆、能交付的系统」。**

**1. Anthropic 核心开发者公开 Claude Code「理解验证」工作流:AI 当老师,人必须能复述**
Claude Code 核心开发者 @trq212 分享、@shao__meng 转述的一套工作流,把 AI 定位成「高效且睿智的教师」——任务完成不算成功,人类对问题、方案、影响能「可复述、可辩护」才算成功。它围绕问题域、方案域、语境域三条轴线,用增量教学 + 用户复述 + 清单测验共 8 个步骤,强制你在长会话里别沦为「审批按钮」。为什么该关心:Tony 自己的规则里就有「验证再声明完成」这一条,而这套工作流等于把同一条纪律产品化成可复用 skill——「用 AI 让人变强」最具体的一版:不是让 AI 替你想,而是逼你真的搞懂。 [source](https://x.com/shao__meng/status/2061612512580346245)

**2. 腾讯混元 Hy-Memory:给长期协作 agent 装一个「第二大脑」**
混元发布了专为 OpenClaw 等长期协作智能体设计的记忆插件 Hy-Memory,基于 6 层记忆框架、System1/System2 双系统与三层进化链;官方数据:记忆数量减少 70% 以上、单条信息密度提升 45% 以上、超长上下文 token 消耗降 35%、记忆更新提速 20%。为什么该关心:长期记忆和自进化系统是 Tony 一直盯的方向。Agent 真正的瓶颈从来不是「这次答得好不好」,而是「下次还记不记得」。**把记忆做成可插拔的一层,等于承认 agent 该有连续的「自我」**——这是它从工具走向系统的关键一步。 [source](https://x.com/TencentHunyuan/status/2061372535267357029)

**3. 用 Claude Opus 4.8 把一本书蒸成一个 AI Skill:45 分钟,不到 20 块**
阿易以《非暴力沟通》为例,给出六步把整本书变成可调用 skill 的流程:靠 Opus 4.8 的 100 万 token 一次性吃下全书,做全局结构分析 + 五类提炼(框架 / 原则 / 技法 / 反模式 / 作者声音),保留书里的原始命名(如 OFNR 四要素),再把触发词改写成「怎么提意见不像在指责」这种大白话。全程约 45 分钟、约 30 万 token,成本不到 20 元人民币。为什么该关心:这是 Skills 生态最接地气的玩法,也是 Tony 的主场——**知识不再只是「读过」,而是被压成一个随叫随到的能力。** 一个人、一下午、二十块钱,就能把一本书变成自己工具箱里的常驻技能。 [source](https://x.com/AYi_AInotes/status/2061419197154857286)

**4. Replit:一句提示词,从零长出一门生意**
Replit 宣布,从今天起一个提示词就能免费产出网站、移动应用、幻灯片和发布视频,还打通了 Stripe、Atlas、QuickBooks、Mercury、doola 等运营工具链。为什么该关心:这正是「一人公司 / MicroSaaS」的极限形态——交付物从「一段代码」升级成「一整套能开张的业务」。当造产品的门槛塌到一句话,真正稀缺的就不再是「会不会做」,而是「做什么、给谁、凭什么有人买单」。**工具越万能,选题和判断力越值钱。** [source](https://x.com/Replit/status/2061534759520760112)

**5. Vista 晒出用 Codex 攒的一堆开源项目:4 插件 + 4 网站 + 5 个 AI Skill**
中文独立开发者 Vista 分享了用 Codex App 等工具做的一批开源项目——4 个 Chrome 插件、4 个网站、5 个 AI Skill(含论文解读、阅读助手),底层用上 GPT-Image-2、Suno,还整合了 Read-frog、Hyperframe 等开源项目。为什么该关心:和上面 Replit 那条对照着看,一个是平台喊出的能力上限,一个是中文创作者已经跑通的真实产出。**「一个人 + AI 编程工具 = 一个小作坊」不再是口号,是已经摆在桌上的成品。** 对想动手的人,这是最好的「别人能、我也能」的样本。 [source](https://x.com/vista8/status/2061443708374208769)

**6. Perplexity「Search as Code」:让 agent 写代码去搜,而不是一个个调函数**
Perplexity 推出面向 AI agent 的新搜索架构 Search as Code——直接生成 Python 代码调用自家搜索栈,取代逐个循环的函数调用,已在 Perplexity Agent API 上线,并成为 Computer 的默认项。为什么该关心:这是 agent 范式的一次小跳变——把「调工具」从一次次离散调用,变成「写一段程序统筹」。**当 agent 开始用代码组织自己的行动,它就从「会用工具的助手」往「会写流程的工程师」挪了一格。** 对琢磨 agent 架构和 MCP 的人,值得细看。 [source](https://x.com/perplexity_ai/status/2061506359326384319)

**7. 阶跃星辰 Step 3.7 Flash:196B MoE,从第一天就为「推理效率」而生**
阶跃发布推理优化型模型 Step 3.7 Flash,196B MoE 架构,用多矩阵分解注意力(MFA)把 KV-cache 成本压到 DeepSeek 的约 22%,又用注意力与 FFN 解耦(AFD)做硬件级高效服务;已上 Fireworks AI,采用 Apache 2.0 许可,可直接拿来搭智能体应用。为什么该关心:它代表另一条路——不靠堆参数,靠「从设计就为效率」来破界。开源 + 低成本 + agent-ready,恰好是想自己掌控、不被 API 价格牵着走的人最在意的三件套。 [source](https://x.com/StepFun_ai/status/2061655529731342402)

**8. 撑起这一切的 Anthropic,悄悄递交了 IPO 草案**
Anthropic 保密向美国 SEC 提交了 S-1 表格草案,为 IPO 拿到了选择权,发行股数与定价待定。公司近期刚完成由 Altimeter Capital 领投的巨额融资、估值逼近万亿美元,并发布了 Claude Opus 4.8。为什么该关心:对几乎整套工作流都跑在 Claude 上的人来说,这不是一条冷冰冰的「投融资数字」,而是「你脚下这块地」要换一种存在方式——从一家私有实验室,走向要对公众市场负责的上市公司。**底座要上市,意味着稳定性、合规、长期主义被摆上台面,也意味着它更难再「随时掀桌」。** 好坏先不论,这是值得 Tony 长期记一笔的生态拐点。 [source](https://www.anthropic.com/news/confidential-draft-s1-sec)
