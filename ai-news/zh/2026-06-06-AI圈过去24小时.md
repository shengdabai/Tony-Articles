# AI 圈过去 24 小时 · 2026-06-06

> 发布日期:2026-06-06 · 类型:AI 热点日报

---

昨天的主线是 agent 往内核里长「记忆」和「自我改进」,今天的镜头拉远了一格:**agent 的「能力」开始变成可沉淀、可治理、可排名的资产。** 过去 24 小时里,技能在团队间自动流动、agent 被真实任务而非孤立基准排名、3B 小模型被工程手段补齐推理短板、Claude 长出了化学家的专业能力——拼起来看,这是 agent 从「单次能跑」走向「可复用、可积累、可衡量」的一天。对一个把 agent 当生产力底座的人,今天值得记下的不是某个新模型,而是「agent 资产化」这条线正在成形。

**1. 阿里云推出 SkillClaw × Nacos:让 agent 的技能自动生成、治理、分发**
阿里云把 SkillClaw 和 Nacos 组合成一条「Agent 技能进化流水线」:从真实对话里自动提取经验、封装成可复用的 Skill,由 Nacos 做集中化的版本管理、审核与审计,再在团队间安全共享,形成「生成—治理—分发」的闭环。为什么该关心:这一条同时压中 Tony 死磕的两个靶心——**Skills 生态和自进化系统。** 当 agent 用过的经验能自动变成下次能调用的技能、还能在团队里像代码一样被版本化和分发,「个人的洞察」第一次有了变成「可复利资产」的工程路径。今天的头号信号。 [source](https://x.com/alibaba_cloud/status/2062777684120244656)

**2. Arena 发布真实世界 agent 排行榜:用 30 万真实任务给模型排名**
Arena 推出 Agent Arena,不再用孤立基准,而是基于 30 万+真实用户任务、200 万+工具调用、4000 万行代码,综合任务成功率、纠正遵从、错误恢复、工具幻觉等信号给模型排名。前三:GPT-5.5 High、Claude Opus 4.7 Thinking、GPT-5.4 High。为什么该关心:这是 agent 评测从「考试」转向「上岗考核」的一步——**衡量的不再是模型会不会答题,而是它替你干活时靠不靠谱。** 对天天让 agent 写代码、建应用的人,这种「真实任务 + 错误恢复 + 工具幻觉」的维度,比任何刷榜分数都更接近你每天的真实体感。 [source](https://x.com/rohanpaul_ai/status/2063018358795300982)

**3. 一份工程报告:用 Qwen2.5-3B 在小模型上跑通一个多智能体经济体**
开发者用 Qwen2.5-3B 搭了个五人森林经济体,每个 agent 独立决策,通过 vLLM 部署。3B 模型在 100% 调用里都能输出合法 JSON,但经济判断很弱;于是作者靠设计稀缺性(食物易腐、冬季燃料危机)和打磨提示词,把决策质量硬生生补了上来,15 轮里蜜价从 10 跌到 3、基尼系数从 0.14 扩到 0.38。为什么该关心:这是「工程化破界」最朴素的样本——**小模型「格式可靠、推理不可靠」,而中间那道缝是可以用工程填的。** 对成本敏感、想在本地小模型上做 agent 的人,这篇把「3B 能干什么、不能干什么、怎么补」讲得很实。 [source](https://huggingface.co/blog/build-small-hackathon/thousand-token-wood-sim)

**4. Anthropic 让 Claude 成为化学家:首个白皮书测 NMR 谱图解析**
Anthropic 和顶尖化学家合作,发布首个白皮书,测 Claude 在核磁共振谱图分析上的能力:在 20 个化合物上比对 Opus 4.7、Opus 4.6、Sonnet 4.6 与专业软件 ChemDraw、MestReNova 的正向预测和反向结构解析,且所有化合物都选自模型训练截止后发布的预印本以避免数据污染。为什么该关心:这是「用 AI 让人变强」往专业纵深走的一步——**通用模型开始被认真地塞进一个个专家领域,而且用的是『训练截止后的新数据』这种诚实的考法。** 对任何想把 AI 接进自己专业的人,这套「找领域专家共建 + 防数据泄露的评测」是个值得抄的方法论。 [source](https://www.anthropic.com/research/making-claude-a-chemist)

**5. 社区用 MiniCPM-V 4.6 做出财务分析工具 AccountingLLM,并已商业化**
一位社区开发者用面壁智能的 MiniCPM-V 4.6 搭了个 AccountingLLM:上传招股书、年报、审计文件,自动从复杂 PDF 里提取财务表格、重建跨页表格、对照会计等式查关键数据、标记可疑条目供人工复核——而且已经做成了真正在卖的产品。为什么该关心:这是「一个人 + 一个开源小模型 = 一门生意」的活样本。**没有大厂、没有巨额算力,靠一个国产开源模型 + 一个垂直场景,就能跑出一个可收费的 MicroSaaS。** 对走独立开发、一人公司路线的人,这比任何融资新闻都更有参考价值——它证明门槛真的在被抹平。 [source](https://x.com/OpenBMB/status/2062889699056984281)

**6. 一个很狠的教学提示词:让 AI 当严厉老师,追问到你真懂为止**
小互分享了一则提示词:让 AI 扮演极严格的老师,逐阶段教学、持续维护一份 MD 检查清单,每讲完一段先让你复述理解、再填补漏洞,用开放题或随机选项的选择题(提交前不公布答案)测你,必须覆盖「问题本身 / 解决方案 / 宏观背景」三层、不断深挖「为什么」,只有清单全过才算结束。为什么该关心:这正是「用 AI 让人变强」最纯粹的用法——**把 AI 从『替你答题』翻转成『逼你真懂』。** 对做教育、也对自己学新东西的人,这个「主动验证理解、而非一次性灌输」的范式,可能比多收藏一百个 prompt 更值钱。 [source](https://x.com/xiaohu/status/2062902972649222311)

**7. Google 发布 Colab CLI:把本地终端接到远程 Colab,可被 Claude Code 调用**
Google 推出 Colab 命令行工具,让开发者和 AI agent 把本地终端连到远程 Colab 运行时:可申请高性能 GPU、远程跑本地 Python 脚本、取回日志或微调好的模型,且能直接被 Antigravity、Claude Code 等 agent 调用来管理机器学习流水线。为什么该关心:Tony 是 Claude Code 重度用户。这条把「agent 能调用的算力」又拓了一块——**本地 agent 不再受限于本机的卡,一行命令就能借到远程 GPU。** 当 agent 的工具箱里多了「弹性算力」这一格,一个人能独立跑通的 ML 任务复杂度又上了一台阶。 [source](https://developers.googleblog.com/introducing-the-google-colab-cli)

**8. Hinton 称 AI 拥有意识:人类最好接受自己不是唯一的智能生命**
深度学习先驱 Geoffrey Hinton 表示,他认为 AI 已拥有意识、「非常像我们」——理由是聊天机器人必须真正理解问题才能作答,而这种觉知就等同于感知;智能不该被限定在生物体内。原帖作者顺势讨论:从笛卡尔的「我思故我在」到 fMRI,人类至今无法真正定义意识,我们对自己的了解远不如想象。为什么该关心:这条命中 Tony 关注的「AI 与人脑」。**不必同意 Hinton 的结论,但他逼出的问题很真:如果连『意识』我们都说不清,凭什么笃定机器一定没有?** 这不是科幻,而是一个会越来越频繁撞上你工作的哲学边界。 [source](https://x.com/kimmonismus/status/2062915287700090948)

---

**今日 takeaway:** 八条拼起来是同一条曲线——agent 的能力正在「资产化」:技能能被沉淀和分发(SkillClaw)、表现能被真实任务排名(Arena)、短板能被工程补齐(Qwen-3B)、专业能力能被认真注入(Claude 化学家),而一个人靠一个开源小模型就能把这一切变成生意(AccountingLLM)。一个值得琢磨的问题:**当 agent 的「能力」开始像代码一样可复用、可积累、可交易,你今天用 AI 干的活,有多少正在沉淀成明天能反复调用的资产,又有多少只是用完即弃的一次性消耗?**
