# AI 圈过去 24 小时 · 2026-06-26

> 发布日期:2026-06-26 · 类型:AI 热点日报

---

今天最值得关心的核心信号是：AI 正在从“聪明模型”继续下沉为可运行的工作系统。Hook、移动端接管、Agent 营销、长期记忆评测和开源编程模型，都在指向同一件事：真正的竞争不只是谁回答得好，而是谁能把上下文、权限、工具、验证和长期反馈闭成循环。

**1. Claude Code Hook 玩法把 AI 变成事件驱动系统**
Claude Code 的 Hook 事件已经可以覆盖权限提醒、开机播报、上下文压缩摘要、文件夹整理、久坐提醒和任务完成推送等场景。为什么值得关心：这不是“多几个自动化脚本”，而是把 AI 从被动聊天框推向事件驱动的个人工作系统。读者真正要学的，是怎样把重复动作沉淀成稳定触发器，让 AI 成为日常流程的一部分。 [source](https://mp.weixin.qq.com/s/LVj2foSXi_hBRKxjuYaUyw)

**2. Claude Code v2.1.193 强化自动模式、遥测和 MCP 体验**
Claude Code 新版本加入 `autoMode.classifyAllShell`，让更多 shell 命令经过自动模式分类器处理，并改进拒绝原因记录、OpenTelemetry 日志、路径自动补全、MCP 认证提示和后台任务稳定性。为什么值得关心：AI 编程工具真正进入工作台后，关键不是“敢不敢自动执行”，而是执行前能不能分类、执行中能不能观测、失败后能不能复盘。工程化破界就在这些细节里。 [source](https://github.com/anthropics/claude-code/releases/tag/v2.1.193)

**3. Codex 正式进入 ChatGPT 移动 App**
Codex 在 ChatGPT 移动端正式可用，并加入设备配对、通知、目标、侧边聊天、文件预览和内联审阅评论等能力。为什么值得关心：这让 AI 编程从“坐在电脑前发指令”变成“人在移动中也能审查、批准、接管后台任务”。跨时空连接的价值很明确：手机负责决策与反馈，开发机负责长时间执行，工作不再被单一设备切断。 [source](https://x.com/OpenAIDevs/status/2070254532911882707)

**4. Runway Agent 2.0 把营销创意变成可测试流水线**
Runway Agent 2.0 面向营销场景，可以在对话中生成活动概念、广告变体、本地化素材、社交内容和下一轮测试创意。为什么值得关心：这代表 Agent 正在进入“创意-发布-数据-再生成”的闭环，而不是只生成单张图或单条视频。对读者来说，启发是把内容生产看成实验系统：AI 不只是帮你做素材，更应该帮你缩短反馈周期。 [source](https://runwayml.com/news/introducing-agent-2)

**5. Ornith-1.0 开源模型家族专攻 Agentic Coding**
Ornith-1.0 覆盖多个参数规模，聚焦智能体编程，在 SWE-Bench、Terminal-Bench、NL2Repo 等基准上给出强成绩，并采用强化学习联合优化任务脚手架与最终解法。为什么值得关心：开源 AI 编程模型正在从“补全代码”走向“自己搭脚手架、跑终端、推进任务”。这会降低本地部署和团队私有化的门槛，也让独立开发者更容易拥有可控的编程助手。 [source](https://huggingface.co/deepreinforce-ai/Ornith-1.0-35B-GGUF/blob/main/README.md)

**6. LongCat VitaBench 2.0 专测长期动态智能体记忆**
VitaBench 2.0 用拟真用户、复杂任务、动态偏好、可执行工具和长时间交互事件，评测智能体在长期个性化场景中的表现。为什么值得关心：长期记忆不是“把聊天记录塞进上下文”这么简单。自进化系统需要知道用户偏好会变、任务会跨天、信息会冲突，而且在不确定时还要主动提问；这类评测会逼 Agent 从短跑演示进入长期陪跑。 [source](https://mp.weixin.qq.com/s/HoiUxYnyJuh2_xdxmg8s8Q)

**7. 用游戏操作数据训练通用智能体的路线继续升温**
一家 AI 公司正在用带按键动作标签的海量游戏数据训练通用智能体，让模型同时学习虚拟环境操作和现实机器人探索。为什么值得关心：游戏数据的价值不只是娱乐内容，而是包含目标、反馈、动作和环境变化的高密度行为记录。用 AI 让人变强的下一层，可能是让模型从人类操作轨迹中学会更稳定的行动策略。 [source](https://techcrunch.com/2026/06/25/from-fortnite-to-robots-general-intuitions-2-3b-bet-that-video-games-can-train-ai-agents-for-the-real-world)

**8. AI 经济报告提醒：收入增长很快，瓶颈也更具体**
一份 AI 经济报告显示，AI 支出和收入形成速度正在快速提升，同时基础设施折旧、电力和数据中心成本仍是关键约束。为什么值得关心：对读者来说，这不是看热闹的融资数字，而是判断机会位置的信号。模型能力越便宜，真正稀缺的就越可能转向可信工作流、分发、行业数据、稳定交付和明确 ROI。 [source](https://x.com/rohanpaul_ai/status/2070288396644491317)

**今日 takeaway：** AI 的主战场正在从“模型多聪明”转向“系统多能闭环”。你现在最该补的，是一个更强模型，还是一套能持续执行、验证、记忆和改进的工作系统？
